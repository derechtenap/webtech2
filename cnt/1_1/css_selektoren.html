<1.1 CSS-Selektoren

(a) Verändern Sie mit jQuery die CSS-Einstellungen, so dass die Hintergrundfarbe aller tr-Elemente hellblau erscheinen.

(b) Modifizieren Sie das Dokument, so dass nur noch jede zweite Zeile mit der Hintergrundfarbe erscheint.

Hinweis: https://www.w3schools.com/jquery/jquery_selectors.asp

Lösung (a)

Um alle tr-Tags der Tabelle mit jQuery zu färben reicht eine einzige Zeile: $('#tabelleA tr').css('background-color', 'lightblue'). Hierbei hat die Tabelle die ID "tabelleA" bekommen (Damit nicht beide Tabellen auf dieser Seite diesen Effekt erhalten).
