2. AngularJS2.1 Aufgabe: Direktiven und Datenbindung

Aufgabenstellung

  1. Schreiben Sie eine Anwendung, die zwei Texfelder "Menge" und "Preis" anzeigt und schon bei der Eingabe den Gesamtbetrag Menge mal Preis ausgibt
  2. Schreiben Sie eine Webseite, in der mit einem Slider ein dargestelltes Bild ausgeblendet werden kann. Benutzen Sie dazu das CSS-Attribut opacity für Ihr Bild.
  3. Schreiben Sie eine Webanwendung, die einen Text und eine Checkbox anzeigt, wobei der Text sich ändert, wenn die Checkbox selktiert ist: Verwenden Sie die Direktiven ng-hide und ng-show und die API https://docs.angularjs.org/api/ng/directive/. (Sollte dies nicht helfen, verwenden Sie die Hinweise auf http://www.w3schools.com/angular/tryit.asp?filename=try_ng_ng-hide).
